<WorkSplitContent subtitle="Keeping the stack focused">
  Because the goal of this project is to experiment with user interfaces and interaction design, I

  lean, fast, and highly developer-friendly.
</WorkSplitContent>

<WorkFeatureList>
  <WorkFeature title="The Foundation (Remix & React)">
    Built on Remix to handle server-side rendering and seamless client-side routing. It provides a
    robust, web-standard foundation that makes managing layout states and data loading intuitive.
  </WorkFeature>
  <WorkFeature title="The Motion (Framer Motion)">
    The engine behind the interactions. Almost every micro-interaction, scroll-linked timeline, and
    page transition is powered by Framer Motion, allowing me to experiment with complex spring
    physics and SVG animations.
  </WorkFeature>
  <WorkFeature title="The Styling (Tailwind CSS)">
    Utility-first styling allows for rapid visual iteration. I rely heavily on custom CSS variables
    and Tailwind configuration to maintain a strict, scalable design system across the entire site.
  </WorkFeature>
</WorkFeatureList>

</WorkSection>
