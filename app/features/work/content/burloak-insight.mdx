  <WorkProse>
    Stepping into my first software engineering co-op at Burloak Technologies was an incredible, eye-opening experience. Surrounded by cutting-edge additive manufacturing, I quickly noticed a common industry challenge: while the manufacturing hardware was incredibly advanced, the data pipelines supporting it were heavily fragmented. 
    
    Data access, storage, analysis, and report generation were largely manual tasks spread across different departments. I was given the opportunity to step in, apply the programming concepts I was learning, and build something that could genuinely help the engineers around me work more efficiently.
  </WorkProse>

<WorkSection>
  <WorkHeader type="FULL-STACK DEVELOPMENT" title="Making data actionable." />
  
  <WorkSplitContent subtitle="From raw metrics to engineering insights">
    Centralizing the data was only the first step. Without an accessible way to view and interpret that data, engineering decisions couldn't be truly data-driven. 
    
    To solve this, I designed and architected a full-stack web application. By building a custom internal dashboard, the engineering team could finally visualize the consolidated data, run historical analyses, and click a button to generate comprehensive reports in seconds instead of compiling them by hand.
  </WorkSplitContent>

<WorkFeatureList>
  <WorkFeature title="The Frontend (React & Tailwind)">
    Built an intuitive, responsive user interface allowing engineers to easily filter, search, and
    visualize complex manufacturing datasets without needing database expertise.
  </WorkFeature>
  <WorkFeature title="The Backend (Node.js & Express)">
    Developed a robust REST API to securely query the MySQL database, handle business logic, and
    format the heavy data payloads for the client.
  </WorkFeature>
  <WorkFeature title="Machine Verification (Power BI)">
    Directed a cross-functional effort to implement a proactive Power BI dashboard. Instead of
    reacting to machine failures, the team could now monitor real-time statistical analysis to
    predict maintenance needs and reduce downtime.
  </WorkFeature>
</WorkFeatureList>
